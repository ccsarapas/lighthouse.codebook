# lighthouse.codebook

The lighthouse.codebook package includes tools for summarizing datasets
used by staff at the [Lighthouse
Institute](https://www.chestnut.org/lighthouse-institute/), the research
division of Chestnut Health Systems.

## Installation

Install lighthouse.codebook by running:

``` r
# install.packages("remotes")
remotes::install_github("ccsarapas/lighthouse.codebook")
```

## SPSS extension

Functionality from this package is also available as an SPSS extension
command [here](https://github.com/ccsarapas/lighthouse.codebook.spss).
